# Laravel Admin Panel

Мощная админ-панель на Laravel 12.x с поддержкой модулей, мультиязычности, Nova-стилем, авторизацией и расширяемой архитектурой.

## Основные возможности
- Модульная система (папка Modules)
- Мультиязычность (модуль Multilang)
- Nova-стиль интерфейса (Tailwind, Alpine.js)
- Управление пользователями
- Логирование и кастомные middleware
- Гибкая настройка через сервис-провайдеры

## Быстрый старт
1. Клонируйте репозиторий:
	```
	git clone https://github.com/ВАШ_АККАУНТ/ИМЯ_РЕПОЗИТОРИЯ.git
	cd ИМЯ_РЕПОЗИТОРИЯ
	```
2. Установите зависимости:
	```
	composer install
	npm install
	```
3. Скопируйте `.env.example` в `.env` и настройте параметры.
4. Сгенерируйте ключ приложения:
	```
	php artisan key:generate
	```
5. Запустите миграции:
	```
	php artisan migrate
	```
6. Запустите сервер:
	```
	php artisan serve
	```

## Структура модулей
- Все модули лежат в папке `Modules`
- Пример: `Modules/Multilang` — мультиязычность
